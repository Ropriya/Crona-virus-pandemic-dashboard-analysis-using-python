# COVID-19 Pandemic Dashboard Analysis

## Overview
A Python-based interactive dashboard for analyzing and visualizing COVID-19 pandemic data including confirmed cases, deaths, recoveries, and trends across different countries and regions.

## Features
- Real-time COVID-19 data visualization
- Country-wise case analysis
- Interactive charts and graphs
- Time series trend analysis
- Comparative statistics
- Global and regional data views

## Technology Stack
- Python 3.x
- Pandas (Data manipulation)
- Matplotlib/Plotly (Visualizations)
- Streamlit/Dash (Dashboard interface)
- NumPy (Numerical computations)

## Installation

```bash
git clone https://github.com/Ropriya/Crona-virus-pandemic-dashboard-analysis-using-python.git
cd Crona-virus-pandemic-dashboard-analysis-using-python
pip install -r requirements.txt
python app.py
```

## Usage
1. Run the dashboard application
2. Select country/region from dropdown
3. Choose date range for analysis
4. View interactive charts and statistics
5. Export data or visualizations as needed

## Dashboard Components

### Key Metrics
- Total confirmed cases
- Total deaths

- Active cases

### Visualizations
- Line charts for trend analysis

- Pie charts for case distribution
